# OXID for OXID API Coding Days 2021

### Requirements
1. Docker with Docker-Compose
2. Composer 2.0
3. webvariants Dev-Stack

### Initialize
Install dependencies: `composer install -n`
Initialize environment: `./docker-init`
Start the container: `docker-compose up -d`
